Back End

Back-end adalah bagian dari website yang mengatur dan mengolah data yang ada di dalam website. Bahasa yang biasa digunakan oleh back-end developer ada PHP, Ruby, dan lain lain. Biasanya, back-end berhubungan dengan koneksi website terhadapa database server. Contoh penggunaan back-end adalah penggunaan website sebagai API untuk digunakan pada suatu aplikasi. Contoh framework yang dapat digunakan sebagai back-end adalah framework Laravel, Express.js dan CodeIgniter.
